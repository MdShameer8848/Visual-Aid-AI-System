# Visual Aid: AI-Assisted Medical Recommendation System

# Project Overview

Visual Aid is a computer vision project that detects wound images and provides first aid recommendations using Artificial Intelligence.

The model classifies images into four categories:

* Burn
* Cut
* Bruise
* Normal Skin

---

# Objectives

* Detect common wound types using image classification
* Provide instant first aid guidance
* Demonstrate AI in healthcare support systems

---

# Model Used
